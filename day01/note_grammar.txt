1. 리엑트만을 위해 사용하는 문법이 있다

- JSX : javaScript.eXtension
- JSX는 자바스크립으와 HTML 문법을 혼합하여 만든 리액트 전용 문법이다.
- 리액트를 이용해서 DOM 요소를 표현하고, 그것을 렌더링할 때 주로 사용한다.
- 자바스크립트로 만든 데이터를 태그 형태로 다루는 것이 기본이다.

2. JSX 문법의 특징

- 태그 형태로 자바스크립트 함수를 표현할 수 있다.
- 자바스크립트 클래스 또는 함수 내부에 HTML 코드를 포함할 수 있다.
- HTML 코드 내부에 자바스크립트 데이터를 포함할 때는 플레이스홀더를 사용한다.
- 자바스크립트 함수가 HTML 코드를 반환할 때는 반드시 하나의 태그로 묶여야 한다.
- 컴포넌트가 컴포넌트를 포함할 수도 있다.


3. JSX 기반 컴포넌트 다루기

- 함수형 컴포넌트에는 속성 객체가 존재한다
- 컴포넌트에 속성을 전달할 때는 사용자 정의 속성을 플레이홀더에 적어준다
- 함수형 컴포넌트에 속성이 전달되면 props 객체가 이를 전달받는다
- style 속성은 JSX에서 객체로 표현된다 중괄호 두겹

  return <div>
